**Grey Lock & Co.** is a premium, high-fidelity geopolitical risk and intelligence platform designed for global businesses, institutional investors, journalists, and sovereign policymakers. Integrating a highly customized military command terminal aesthetic (inspired by *Palantir*, *Bloomberg Terminal*, *Stratfor*, and *Anduril*), the platform provides real-time situational feeds, historical intelligence archives, and forward-looking strategic forecasts focusing heavily on the South Asian borderlands and global maritime corridors.

---

## 🌟 Visual Excellence & Product Vision

Grey Lock & Co. is engineered to answer four critical questions for decision-makers:
1. **What problem do we solve?** We synthesize chaotic global events into reliable, open-source intelligence signals, separating deliberate state-aligned noise from true operational risk.
2. **Why are we different?** We deploy strict, cross-verified **Analysis of Competing Hypotheses (ACH)** methodology combined with dynamic automated telemetry monitoring (spectrum, transponder, and narrative scans).
3. **Can users experience the intelligence platform?** Yes, via an interactive command-line style HUD, featuring live telemetry logs, responsive country matrices, and visual SVG tracking.
4. **Can we capture leads?** Seamlessly, utilizing secure, TLS-encrypted corporate briefing channels and verified PGP keys for highly sensitive inputs.

## 🛠️ Architecture & Premium Features

### 🖥️ 1. Geopolitical HUD Terminal (`/platform`)
A state-of-the-art operational workspace mimicking cyber-defense systems:
* **Interactive Threat Map**: A custom SVG world centerpiece tracing maritime chokepoints, naval routing, and active border tensions (Delhi/Kashmir, South China Sea, Hormuz Strait, Black Sea, Myanmar corridor) with absolute responsive scaling.
* **Live Alerts Feed**: Interactive scrolling SIGINT signals with dynamic search filtering and telemetry logs.
* **Risk Coefficient Table**: High-contrast, sorting operational matrices detailing regional escalations.
* **SVG Sparkline Visualizations**: Custom, mathematically balanced mathematical indicators showing risk indices over a 7-day trailing average.

### 📚 2. Institutional Reports Library (`/reports`)
High-fidelity strategic briefs written by veteran analysts, categorizing regional friction points:
* Instantly searchable by title, author, and intelligence code.
* Quick filters for **Geopolitics**, **Security**, **Information Warfare**, **South Asia**, and **Global**.
* Simulated automated cryptographic downloads with elegant UI-state triggers.

### 📰 3. Editorial Insights Magazine (`/insights`)
Premium long-form forecasts blending the narrative depth of *Foreign Policy* and analytical accuracy of *Stratfor*:
* Features a prominent editorial highlight card detailing tactical grey zone proxy operations.
* Modular, multi-column articles with citations, publication dates, and author cards.

### 🌐 4. Interactive Universal Light Theme
* An elegant **Theme Switcher** mapped directly to system-preference triggers and client-side `localStorage`.
* **Daylight Print Slate style** designed to preserve elite typography layout, while providing ultra-legible contrast (slate-white card surfaces, charcoal-blue headers, high-contrast gold indicators, and zero washed-out white text blocks).

## 🚀 Quick Setup & Installation

### 1. Prerequisites
Ensure you have [Node.js (v18.x or later)](https://nodejs.org/) and `npm` installed.

### 2. Clone the Repository
```bash
git clone https://github.com/your-username/grey-lock-co.git
cd grey-lock-co
```

### 3. Install Dependencies
```bash
npm install
```

### 4. Run Development Server
```bash
npm run dev
# Server will launch locally at http://localhost:3000 (or http://localhost:3002 if 3000 is occupied)
```

### 5. Production Build
Ensure strict TypeScript type-checking and optimization passes successfully:
```bash
npm run build
```
